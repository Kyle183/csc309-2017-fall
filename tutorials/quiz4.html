Quiz #4 - SOLUTION

Assume you have been working with your teammates on a repository called "QUIZ4". Enter the sequence of commands that will let you do the following:

  1. Obtain the code your teammates are working on.
  2. update your working directory with the latest work from everyone
  3. add an empty file called feature.txt
  4. update the remote repo with your changes

Pick from the following list of commands and use ONLY the ones you need and put them in the fields below IN THE CORRECT ORDER

  • git commit -am "something"
  • git fetch QUIZ4
  • git push QUIZ4
  • git pull QUIZ4
  • git status
  • touch feature.txt
  • git add feature.txt
  • git log
  • git merge QUIZ4
  • git commit -m "something"

Solution

  • git pull QUIZ4
  • touch feature.txt
  • git add feature.txt
  • git commit -m "something"
  • git push QUIZ4
